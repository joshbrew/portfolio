Libraries

Computation/Performance

magicworker Convenient webpacked web worker utilities, including writing arbitrary functions to threads, transferring classes, worker-based canvas and threejs rendering, piping, proxies, and more.
A version of this is in our liveserver library as well for creating front and backend threaded server applications. Garrett Flynn did the webpacking and lots of testing.

gpujsutils Write GPU kernels in js!
Wrapper with a stable webpacked build of the awesome gpu.js library. It integrates some utilities with premade functions and makes it easy to write persistent, dynamic kernels or canvas kernels to the gpu without messing with gpu.js as much.


Scripting & Data

acyclicgraph Easy async graph node operation tree scripting utility.
Featured in this webapp to handle the physics tree.


anotherstatemanager Multi-tiered state manager including fast checking large objects/arrays for animation
A lot of work was put into this with Garrett Flynn to make it appropriate for large real time dataset animations.


anothersequencer Simple function sequencing with async and animations.


objectlisteners This is an arbitrary object listener for checking large objects and specific properties for changes in a loop e.g. for animation updates, it forms the base of my state management library.


brainsatplay-data Data Structures & Algorithms for Brains@Play environment. Working repo.
brainsatplay-storage Storage functions for Brains@Play. Includes minimongo, browserfs, google drive, & csv utilities.

Visual

fragelement Minimally abstracted HTMLElement for responsive web components and easier scripting.
Used it to write this application it super fast with raw html/js/css!


threeshaderhelper Wrapper for conveniently editing THREEShaderMaterials and properties.
Automatically generate sliders in a GUI from uniform declarations detected in the glsl shader text. Made with Garrett Flynn. Used for biofeedback in Brains@Play, idea suggested by Stephanie DuPont at the Biofeedback Institue of Los Angeles.


dynamicparticles Simple particle rendering class. I even made a multithreaded example for it with magicworker.
Includes a fairly optimized boids algorithm (you can see orbiting planet javascript) using limited a random proximity search where the behaviors remain consistent even checking only one random neighbor per frame in large groups. I intend to expand on this over time.


webglplotutils A simple stacking/resizing utility for time series data with webgl-plot's line plot utilities.
It performs interpolation automatically if you supply variable sized data sets, thought at the expense of performance.
Consistent data sets can be rendered at millions of lines per frame at 60fps, though with minimal styling. ThreeJS or something is better for styling. Smoothiejs or uPlot are great minimal/performant plotting libraries, too, with different tradeoffs.


brainsatplay-math A bunch of math for linear algebra, 2d and 3d transforms, covariance, cross-correlation, calculus, etc. Some e.g. the power iteration is borrowed from other FOSS libs and rewritten for clarity or generalization (with sources).


Server

liveserver-router
liveserver-frontend
liverserver-backend 		Typescript library made with Garrett Flynn that handles a ton of abstraction for real time servers, including user databases and streaming with peer2peer (incl. webrtc) or dedicated servers, dynamic routing, web workers (which can run their own servers), mongodb hooks, and more.


Drivers

biquadjs Feed-forward digital biquad filters for real time signal processing. Very common and useful filtering scheme.
Translated from an arachnoid.com python tutorial while unsuccessfully trying to build a phase-locked loop.


hegduinojs USB Serial, Bluetooth LE, and WiFi Event Source drivers for the HEGduino ESP32-based device.


cytonjs USB Serial drivers or the OpenBCI Cyton. Includes rollover buffers for the latest data.


freeeeg32js USB Serial drivers for the FreeEEG32. Includes rollover buffers for the latest data.


BreathCapture Audio capture with basic smoothed peak-finding based breath detection and phase locking. It works like half of the time but it works nonetheless. Made for the 2020 XR Brain Jam by Games For Change with a VR demo.


Chrome Serial Utils Simple Serial USB utilities for chrome extensions (which had their own serial port library pre-chromium integration).

More Info

Javascript is currently my primary software engineering language. It holds some of the most promise as it continues to gain more updates every year as the base of all web software, and as more people see the value in open, unified, interoperable frameworks. My hope is that as more mature web frameworks like WebGPU come along, we'll see more browser engines that are lighter weight and easier for coding frontends. This brings the cost of development down significantly so a larger shared creative engineering space can continue to open up and reach larger audiences.

My favorite thing about javascript has to be that it helped me grasp systems engineering so much better than I could with C/C++ or Python, etc. That's because it's more playful and designed for boneheads like me to combine fairly complex features like Bluetooth, audio, graphics, etc. into a frontend with good cross-platform performance. It's getting easier to build flexible, high performance applications with minimal code, something I had given up on trying to do for several years until these past few. Now I can say I enjoy doing this, if this site is any evidence.
