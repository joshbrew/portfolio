NodeJS Portfolio


Applications built with NodeJS (all pre-alpha demos)
Brains@Play PWA Real time sensing platform for open EEG, FNIRS, eye tracking, and more hardware co-developed with Garrett Flynn and a little help from our small but growing community. This is a long term project. Features tileable applications with dynamically loaded resources, multiplayer, device connectivity, data visualization, multithreading, etc.
Portfolio App This app takes the esbuild + python base and uses it for practical demonstration. It serves python in production with hypercorn, and the PWA features are demonstrated with an https server. The server-server and server-client communications simply trigger a radio animation to visualize the process occurring in a fun way.
HEGAlpha PWA Simple webapp for my HEGduino project. This app is so small it can serve from an ESP32 web server!
EEG PWA Pre-Brains@Play web demo built with help from Dovydas Stirpeika. Features GPUjs FFTs to analyze FreeEEG32 data in the browser in real time!.
esbuild App Base esbuild is the latest build tool for web applications. It's fast and easily customizable for js, ts, react, etc. It's easy to step into a production setting and not overburden your dev tools. Includes custom hot reloading and PWA presets.

App Base		 esbuild app base plus my web components library! Includes custom hot reloading and PWA presets.

App Base		 A stripped down base for creating wicked fast web applications with esbuild, Demonstrates dual server support for building webapps with NodeJS and data streaming with Quart's ASGI framework.
